# AI Meal Prep Planner (Python)

This repository was converted to a minimal Python project scaffold.

Quick start

1. Create a virtual environment:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Run the CLI:

```bash
python -m ai_meal_prep_planner --help
```

3. Run tests:

```bash
pytest -q
```
